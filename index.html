About the Theme

This purple theme, called "Lande della Discordia", was created by @ungiglio to customize his own Discord client. Now, he's sharing it with everyone who wants to add a touch of elegance to their Discord interface.

The theme features a beautiful purple shade (#b266ff) that transforms your Discord experience into something truly unique.

Installation [vencord.dev version]

To install the "Lande della Discordia" theme, follow these steps:

  1. Download the theme files from our GitHub repository
  2. Open your Discord client
  3. Navigate to Vencord > Themes
  4. Under "Local Themes", select "Open Themes Folder" and upload the downloaded files named "discordia.theme.css"
  5. Click "Load Missing Themes" and check the toggle near "Lande della Discordia" card

Installation [betterdiscord.app version]

To install the "Lande della Discordia" theme, follow these steps:

  1. Download the theme files from BetterDiscord
  2. Open your Discord client
  3. Navigate to Settings > Themes
  4. At the top of the themes page, click the "Open themes folder" button
  5. Drag the file you downloaded into the themes folder that opened
